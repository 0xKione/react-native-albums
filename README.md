# react-native-albums

## Set up
### Install Dependencies
```sh
$ cd path/to/repo
$ npm install
```

### Install ESLint and rules
```sh
$ npm install -g ESLint
$ cd path/to/repo
$ npm install --save-dev eslint-config-rallycoding
```

### Install ESLint in VSCode
Open VSCode
Press Ctrl + P
Type 'ext install vscode-eslint'
Update and Enable it
